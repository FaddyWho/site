UX/UI

Snippets.

Year

2024

Project Type

App Design

Duration

5 weeks

Overview

Snippets is a supplementary language learning app, with a focus on vocabulary. It uses augmented reality and artifcial intelligiance to help users identify objects in their proximity and learn the the words for it.
Users can also contribute to the vocabulary database by tagging objects in the language they know and review tags contributed by other users to help expand the vocabulary database and improve the AI recognition.

Problem and Solution

Learning vocabulary is difficult and users often lose interest in standardised courses which arent catered to their lifestyle and needs.

Design an app that uses modern technologies like AI and AR to create custom learning experiences based on each users lifestyle and proximity to objects they use everyday.

Figma

Role and Process

I served as the User-experience and User-interface designer for the app.

System

The use of AI is based in points the user can earn and spend, which in turns improves the AI in a feedback loop.

AR Navigation

Objects highlighted in blue are to be selected while object in yellow are next in line based on proximity to the one hovered one.

Once an object is selected, the highlight color will turn to green, and users can go inside the object to select smaller parts of it which is controlled by different arrows.
